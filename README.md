# Detective Mystery Game (C++)

A text-based detective mystery game developed using C++ and Object-Oriented Programming concepts.

## Project Overview

This project allows players to step into the role of a detective and solve a mystery by collecting clues, analyzing evidence, and making decisions throughout the investigation.

## Features

* Interactive text-based gameplay
* Multiple suspects and clues
* Decision-based investigation flow
* Object-Oriented Programming implementation
* User-friendly console interface

## Concepts Used

* Classes and Objects
* Encapsulation
* Functions
* Loops
* Conditional Statements
* Arrays / Data Structures
* File Handling (if used)
